Get Started

Install

One small step.

This is still a work-in-progress, and there are many things that don't work!

Pantheon is a mostly distro-agnostic, and work by many talented people over the years has made using the desktop outside of elementaryOS possible, but its some way from being complete. Some built-in apps and settings items may not work properly, and entirely broken features have been removed. However, it is usable for day-to-day activities and every effort has been make sure the OS is stable and safe.

See sodaliterocks/sodalite ➔ Status for more information.

  1. Install an OSTree version of Fedora, such as Fedora Silverblue.
  2. Open a terminal and issue these commands:
    • sudo ostree remote add --if-not-exists sodalite https://ostree.sodalite.rocks --no-gpg-verify
    • sudo ostree pull sodalite:sodalite/stable/x86_64/stable
    • sudo rpm-ostree rebase sodalite:sodalite/stable/x86_64/stable
  3. Stick the kettle on and make yourself a cuppa. It'll take a while.
  4. Reboot when prompted. Sit back in awe as the desktop loads up.
    • Updates will occur automatically if you update everything from Software (which runs in the background by default and sends desktops notifications).
    • As they are installed via Flatpak, various GNOME apps will still be lingering. Remove these with sudo sodalite-uninstall-gnome-apps.

Need more help? See sodaliterocks/sodalite ➔ Getting.
